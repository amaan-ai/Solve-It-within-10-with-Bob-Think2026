Getting Started: Explore Bob's creative capabilities beyond traditional coding. Build a game and see what Bob can create!

Build a Brick-Breaker Game Agent Mode

Build a simple classic arcade-style brick-breaker game similar to DX-Ball that I can easily launch and play on this system. Requirements: * Player starts with 3 lives; after all lives are lost, show a "Game Over" screen * Include smooth controls, modern visuals effects fun gameplay feel * The game should be easy to run on macOS, ideally by double-clicking a launcher file * Include clear setup/run instructions * Keep the implementation simple and lightweight * Score tracking and Highest score so far * Pause/resume feature
Ready to Play: Once Bob creates the game, you can launch and start playing!
Take-Home Challenge:
Try adding more features to the game, e.g.
  • Multiple levels
  • Power ups / upgrades: Increased paddle width, Multiple balls, Increased ball strength/speed

Or try building a different game with Bob. How about Super Mario? Experiment with different game genres and see what Bob can create!
What to Expect:
  • Interactive game development with various genres
  • Creative problem-solving approaches
  • Rapid prototyping and iteration

Understand the Dataset Ask Mode

What is this sample-data.csv about? Give me quick exploratory data insights

Analyze Dataset and Generate Visualizations Agent Mode

Analyse @sample-data.csv and produce an executive-ready HTML dashboard. Use multiple agents. The dashboard (dashboard.html) should include: - An executive summary at the top: 3-5 key takeaways as bullet points - KPI headline cards (top-line numbers that matter most) - A trend chart over time (if a date column exists) - A category/segment bar chart - Use only inline HTML, CSS, and vanilla JavaScript — no external CDN links or library imports. Draw charts using SVG or the Canvas API directly. - Style with IBM Carbon Design System conventions: IBM Plex Sans font (loaded via a single Google Fonts @import in the <style> tag), Carbon colour tokens (blues/grays), clean card layout. No emoji anywhere.
What You'll Get: Bob will spin up multiple agents to analyse the data and build the dashboard — delivering a fully self-contained HTML file you can open straight in the browser with no internet required.
Take-Home Challenge:
Try these advanced analysis tasks:
  • Ask Bob to perform predictive modeling or clustering analysis
  • Request specific business insights based on your domain
  • Have Bob create an interactive dashboard with filters and drill-down capabilities
  • Ask for automated report generation with key findings
What to Expect:
  • Data analysis and visualization from CSV/Excel files
  • Creative problem-solving approaches
  • Rapid prototyping and iteration
