Analyze Dataset and Generate Visualizations

I have a dataset sample-data.csv that I'd like analyzed for a business executive audience. Please: 1. Quickly scan the data — note what it covers, the date range if any, and flag any data quality issues worth knowing about. 2. Analyze it for: - Key performance trends over time (if the data has a date/time dimension) - How performance breaks down across the main categories/segments in the data - Relationships between key metrics (e.g., what drives what) - Any notable outliers, anomalies, or standout results 3. Build an interactive HTML dashboard with: - A top-line KPI summary (a few headline numbers that matter most) - Trend chart(s) over time, if temporal data exists, at a sensible time granularity - Category/segment comparison charts (bar charts) - One chart showing the relationship between key metrics - A correlation heatmap only if it adds clear business value — otherwise skip it - Other important data visualization if any 4. Add a short executive summary at the top: 3-5 bullet points on key takeaways, and 2-3 recommended next steps or areas worth digging into further. Use Chart.js and/or Plotly. Style with the IBM Carbon Design System — IBM Plex Sans typography, Carbon's color tokens and layout conventions. Keep it clean, professional, and easy to skim — prioritize clarity over statistical detail. No emoji or emoticons anywhere.
What You'll Get: Bob will produce an executive-ready HTML dashboard with KPI headlines, trend and comparison charts, key takeaways, and recommended next steps — all styled with IBM Carbon Design System conventions.

Voice Singing Prompt

I have a voice sample @my_voice_sample.wav. Using only lightweight, offline Python libraries (no XTTS, no RVC, no model downloads, no Terms of Service prompts), generate a ~20-second audio clip of the opening of "As It Was" by Harry Styles sung in a voice that matches the pitch and tone of my sample. Use this exact approach: 1. Use pyttsx3 (offline TTS) to synthesise the opening lyrics as spoken audio → spoken_lyrics.wav 2. Use librosa to analyse the fundamental frequency (F0/pitch) of my_voice_sample.wav 3. Pitch-shift spoken_lyrics.wav to match that F0 → pitched_voice.wav 4. Use pydub to generate a sine-wave chord backing track in A major (~20 sec) → backing_track.wav 5. Mix pitched_voice.wav + backing_track.wav, normalise volume → as_it_was_my_voice.wav 6. Print the file path and the command to play it (afplay on macOS / aplay on Linux) Only use: pyttsx3, librosa, soundfile, pydub, numpy. pip-install anything missing. No downloads over 50 MB.

Bob will produce as_it_was_my_voice.wav in your working directory in ~1–2 minutes. Open it to listen!

Advanced Prompt — Bob Composes in Your Voice

Using @my_voice_sample.wav as the reference voice, generate a ~20-second audio cover of "As It Was" by Harry Styles (2022). Instructions: - Match the vocal timbre, tone, and character from the reference voice file - Use open-source libraries such as Coqui TTS (XTTS v2), so-vits-svc, or RVC for voice matching - Synthesise or use a royalty-free instrumental backing track for "As It Was" - Render the final output as a single mixed audio file: as_it_was_my_voice.wav - Print the steps to play the generated file locally once done
